## Description
+ master是对接飞书机器人
+ template是nest项目模板

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

### nest模板
+ 在template分支下是整理好的nest模板分支
+ 数据连接
+ 异常处理

+ 拦截器

+ 全局字段校验
+ 暂时不要使用热更新
